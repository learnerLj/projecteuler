# projecteuler

项目结构：
- src/：题目源代码目录。
- app/：应用程序入口点。
- test/：测试代码目录。
- projecteuler.cabal：Cabal 文件，描述了项目的依赖项和配置，一般自动生成，不要动。
- stack.yaml：Stack 配置文件。
- package.yaml: 
- Setup.hs：用于 Cabal 配置的文件。
